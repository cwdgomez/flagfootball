Service Overview
🌐
GitHub Pages
Hosts your static PWA files. Free. Served via HTTPS. Auto-deploys when you push to main.
💳
Stripe
Handles all payments. $9.99/mo or $79.99/yr. Sends webhooks on every billing event. Never touches your servers.
⚡
Vercel
Runs your two serverless functions. Receives Stripe webhooks and validates license keys on demand. Free tier.
🗄️
Supabase
PostgreSQL database storing license keys, emails, subscription status, and billing period. Free tier. Row-level security enabled.
📨
Resend
Sends the license key delivery email from pro-version@itcc.llc. Triggered automatically on every new subscription. Free tier.
📱
CGMax FFTP App
PWA on GitHub Pages. All coaching data stays on-device (localStorage). Only license key is sent to backend for validation.
Monthly cost at current scale: ~$0. All services run on free tiers. Stripe charges 2.9% + 30¢ per transaction (deducted from revenue automatically). First paid customer is pure profit minus Stripe fees.
